# Herlin AI Assistant - Frontend

React-based frontend for the Herlin AI Assistant platform.

## Features

- Modern chat interface with Herlin AI
- Dark theme with gradient backgrounds
- Responsive design
- Real-time messaging UI
- Conversation sidebar
- Loading indicators

## Quick Start

Open `index.html` in a browser or serve it with any static server:

```bash
npx serve .
```

## Structure

```
frontend/
├── index.html       # Main HTML file with React app
├── package.json     # NPM dependencies
```

## Tech Stack

- React 18
- TailwindCSS
- Babel
- Vanilla CSS animations
